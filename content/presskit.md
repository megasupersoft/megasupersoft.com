# Press Kit

Everything you need to write about MegaSuperSoft.

## Company overview

**MegaSuperSoft** is an independent software company based in New Zealand. We build local-first, open-source desktop software and games.

- **Founded:** 2024
- **Location:** New Zealand
- **Website:** [megasupersoft.com](https://megasupersoft.com)
- **Email:** hello@megasupersoft.com

## What we do

We make two kinds of things:

**Software** — Open-source, local-first desktop applications. Your data stays on your machine. Tools for finance, VFX, creative work, and file management.

**Games** — Original titles spanning roguelike arcade, adventure RPG, and isometric exploration.

## Products

### FFFFinance

Local-first personal finance for New Zealand. Bank-connected, AI-powered, 20+ themes.

Desktop application connecting to NZ bank accounts via Akahu, featuring local AI-powered transaction categorisation and 19 customisable dashboard panels — all data remains on-device.

- **Website:** [ffffinance.org](https://ffffinance.org)
- **Press kit:** [ffffinance.org/presskit](https://ffffinance.org/presskit)
- **GitHub:** [DangerDrome/FFFFinance_app](https://github.com/DangerDrome/FFFFinance_app)
- **Platforms:** macOS (Apple Silicon + Intel), Windows, Linux
- **Pricing:** Free / Pro $9/mo / Cloud $19/mo
- **Tech:** Electron, React, TypeScript, SQLite, node-llama-cpp
- **AI:** Local LLM inference (Qwen3), optional cloud fallback
- **Bank integration:** Akahu (NZ open banking)

## Brand guidelines

- Use the dark logo on dark backgrounds, light logo on light backgrounds.
- Do not stretch, rotate, or alter the colours of the logo.
- Maintain clear space around the logo equal to at least the height of the "MEGA" text.
- The ASCII art style is a core part of the brand identity. Preserve the monospace rendering.
- The name is one word: **MegaSuperSoft**. Capital M, S, S. No spaces, no hyphens.
- Product names: **FFFFinance** (four F's, one word), **OpenComp**, **NodeUI**, **FileUI**.
